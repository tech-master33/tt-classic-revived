# tt-classic-revived

The Angels Clan website — a companion web page for the tt-classic-revived TeamTalk community. Hosted via GitHub Pages.

**Live site:** [tech-master33.github.io/tt-classic-revived](https://tech-master33.github.io/tt-classic-revived)

## What it is

tt-classic-revived is a web presence for the Angels Clan TeamTalk community, providing:

- Community introduction and information
- Feature highlights
- Contact information for joining the TeamTalk server

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page |
| `styles.css` | Stylesheet |
| `_config.yml` | Jekyll / GitHub Pages config |

## Making changes

1. Fork this repo
2. Edit `index.html` or `styles.css`
3. Open a pull request — GitHub Pages will deploy the new version automatically when merged

## Accessibility

All changes to `index.html` should follow basic web accessibility guidelines:

- All images must have `alt` text
- Links must have descriptive text (not "click here")
- Headings must be in order (`h1` → `h2` → `h3`)
- Colour contrast must meet WCAG AA
