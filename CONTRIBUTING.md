# Contributing to tt-classic-revived

Thank you for helping improve the Angels Clan website.

---

## Ways to contribute

1. Fix a typo or broken link in `index.html`
2. Improve the layout in `styles.css`
3. Add new content sections
4. Report accessibility issues

---

## Step 1 — Fork

1. Open github.com/tech-master33/tt-classic-revived
2. Activate Fork → Create fork

---

## Step 2 — Clone

```bash
git clone https://github.com/YOUR-USERNAME/tt-classic-revived.git
cd tt-classic-revived
```

---

## Step 3 — Make changes

Edit `index.html` or `styles.css` in a text editor.

### Accessibility checklist

Before submitting:
- All images have meaningful `alt` attributes
- All links have descriptive text
- Headings are in correct order
- Text contrast meets WCAG AA

---

## Step 4 — Push and pull request

```bash
git add .
git commit -m "fix: update contact link"
git push origin main
```

Open a pull request on GitHub. GitHub Pages deploys automatically when merged.

---

## Reporting a bug

Open github.com/tech-master33/tt-classic-revived/issues and describe the issue.
